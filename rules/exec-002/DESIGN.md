# Rule Design: exec-002 (script-path-valid)

## 基本信息

| 字段 | 值 |
|------|-----|
| **Rule ID** | exec-002 |
| **名称** | script-path-valid |
| **验证维度** | 可执行性验证 |
| **严重级别** | Warning |
| **适用阶段** | Phase 4 |

---

## 设计原理

### 为什么这条规则重要

Skill 目录下的 scripts/ 文件夹包含可执行脚本。如果脚本路径无效，Skill 的自动化能力会受影响。

### 目录结构规范

```
skill-name/
├── SKILL.md
├── scripts/        # 可选
├── references/     # 可选
└── assets/         # 可选
```

---

## 实现逻辑

### 检测方法

```
1. 检查 scripts/ 目录是否存在（如果 SKILL.md 引用了脚本）
2. 验证引用脚本的路径是否有效
```
