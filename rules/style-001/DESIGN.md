---

## 设计原理

### 为什么这条规则重要

代码块如果没有指定语言，AI 阅读时无法进行语法高亮，影响可读性。指定语言后，代码展示更清晰，也便于 AI 正确识别代码类型。

### 示例

```
# ✅ 正确
```javascript
function hello() {
  console.log("Hello");
}
``

# ❌ 错误
```
function hello() {
  console.log("Hello");
}
``
```

---

## 实现逻辑

### 检测方法

```
1. 查找 markdown 中的代码块标记 ```
2. 检查代码块是否紧跟语言标识符
3. 如果没有语言标识符，报告 Hint
```

### 代码块模式

```
/```(\w*)\n/  — 匹配 ``` 后换行的情况
/```(\w*)$/  — 匹配 ``` 在行尾的情况
```
